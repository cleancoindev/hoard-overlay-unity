◆ OnStartAuthority()

override void AuthSpawnableObject.OnStartAuthority ( )
inlinevirtual

This is invoked on behaviours that have authority, based on context and 'NetworkIdentity.localPlayerAuthority.'

This is called after OnStartServer and OnStartClient.

When NetworkIdentity.AssignClientAuthority

is called on the server, this will be called on the client that owns the object. When an object is spawned with NetworkServer.SpawnWithClientAuthority, this will be called on the client that owns the object.

Reimplemented from UnityEngine.Networking.NetworkBehaviour.

◆ OnStopAuthority()

override void AuthSpawnableObject.OnStopAuthority ( )
inlinevirtual

This is invoked on behaviours when authority is removed.

When NetworkIdentity.RemoveClientAuthority is called on the server, this will be called on the client that owns the object.

Reimplemented from UnityEngine.Networking.NetworkBehaviour.
